What is the document object model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents.

What is the event delegation in the context of the DOM, and why is it useful ?

Event delegation is a technique in JavaScript for handling events efficiently by leveraging the event bubbling mechanism in the DOM.
